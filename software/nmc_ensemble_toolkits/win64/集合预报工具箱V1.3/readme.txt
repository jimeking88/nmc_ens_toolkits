;
; 简要安装说明
;
1、在配置文件中设置集合预报数据存放地址, 再根据屏幕的分辨率调节窗口比例参数, 也可调整其他一些显示参数, 如地图的显示范围, 地图边界等.
2、安装resource目录下的'systemdll32_setup.exe'或'systemdll64_setup.exe'(如果本机上安装过的，则无需再次安装)
3、双击'集合预报工具箱1.0.exe'运行即可
